Html cheatsheet

Basic Html Elements

Elements Attributes
Element name Tag/Tags Description Attribute name Description
Paragraph <p> Paragraphs are blocks of text separated from adjacent blocks by blank lines. This element only includes the global attributes.
Headings <h1> to <h6> Headings are titles or subtitles. <h1> is the biggest and <h6> is the smallest. This element only includes the global attributes.
Anchor <a> This elementcreates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else. href The URL that the hyperlink points to.
Image <img> The image element embeds an image into the document. alt Defines an alternative text description of the image.
src This sets the url of the element.
Ordered list <ol> This element element represents an ordered list of items.Each list item starts with the <li> tag. type Sets the numbering type:for lowercase letters A, for uppercase letters i, for lowercase Roman numerals I, for uppercase Roman numerals 1, for numbers (default)
Unordered list <ul> This element element represents an unordered list of items.Each list item starts with the <li> tag. type This attribute sets the bullet style for the list: circle, disc, square
List <li> This is used to represent an item in a list. It must be contained in a parent element: an ordered list, an unordered list , or a menu. type This attribute sets the bullet style, or the numbering type for the list and will override the one used by its parent.
